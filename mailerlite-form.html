Option A — MailerLite Embedded Form (easiest)

  1. Login to MailerLite → Forms → Create embedded form.
  2. Build the fields (name, email, beta checkbox) in the MailerLite form builder or keep the default email-only form.
  3. Copy the embed code and paste it directly into your `index.html` in place of the current <form id="signup"> block.

Option B — Custom HTML form + serverless endpoint (secure, flexible)

Use this if you want to keep your MailerLite API key secret and tag beta signups. Deploy the serverless function (see mailerlite-serverless.md) and point the form at /subscribe (or whatever path your provider uses).

Replace the existing <form id="signup"> in your `index.html` with this form HTML and include the script shown below.

<form id="signup" class="signup-form" action="#" method="post">
  <input id="name" name="name" type="text" placeholder="Full name" required>
  <input id="email" name="email" type="email" placeholder="Email address" required>
  <label><input type="checkbox" id="beta" name="beta" value="yes"> Yes — I want to be a beta tester (optional)</label>
  <button type="submit" class="btn btn-primary">JOIN THE PRIORITY LIST</button>
</form>

<script>
document.getElementById('signup').addEventListener('submit', async function(e){
  e.preventDefault();
  const name = document.getElementById('name').value;
  const email = document.getElementById('email').value;
  const beta = document.getElementById('beta').checked ? 'yes' : 'no';

  try{
    const res = await fetch('/.netlify/functions/subscribe', { // change path if using another provider
      method: 'POST',
      headers: {'Content-Type':'application/json'},
      body: JSON.stringify({ name, email, beta })
    });

    if (res.ok) {
      // either redirect to thankyou page or show inline message
      window.location.href = 'thankyou.html';
    } else {
      const text = await res.text();
      alert('Signup failed: ' + text);
    }
  } catch (err) {
    console.error(err);
    alert('Network error. Please try again later.');
  }
});
</script>
